# 有給申請システム（社内DX）

## 概要
Google Apps Script（GAS）とスプレッドシートを使った有給申請・管理アプリです。

社員が申請し、管理者が承認・否認を行えるシンプルなシステムを目指しています。

---

## 使用技術
- Google Apps Script（GAS）
- Googleスプレッドシート
- HTML / CSS / JavaScript

---

## 現在できること
- 有給申請の登録
- 管理画面で申請一覧の表示
- ステータス管理（pending / approved / rejected）
- 承認・否認処理
- ログ表示
